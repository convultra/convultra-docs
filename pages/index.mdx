## Quick Start

Get started in 5 minutes:

### 1. Add the SDK

```html copy
<script 
  src="https://cdn.convultra.com/ultra.min.js"
  data-convultra-api-key="YOUR_API_KEY"
  data-convultra-endpoint="https://your-endpoint.com"
></script>
```

### 2. Track Conversions

```javascript copy
// Track a purchase
Convultra.trackPurchase({
  orderId: 'ORD-12345',
  value: 99.99,
  currency: 'USD'
})
```

### 3. Connect Platforms

Go to **Dashboard â†’ Integrations** and connect your ad accounts.

That's it! Conversions are now forwarded to all connected platforms.
