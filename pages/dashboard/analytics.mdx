# Analytics

Convultra provides Plausible-style analytics that respect privacy while giving you actionable insights.

## Overview Metrics

The main dashboard shows:

| Metric | Description | Calculation |
|--------|-------------|-------------|
| **Unique Visitors** | Distinct users | Fingerprint + session dedup |
| **Pageviews** | Total pages viewed | Count of `pageview` events |
| **Bounce Rate** | Single-page visits | Visits with 1 pageview / Total visits |
| **Avg. Duration** | Time on site | Session end - Session start |

---

## Traffic Sources

### Channels

See where your traffic comes from:

| Channel | How Identified |
|---------|----------------|
| **Direct** | No referrer |
| **Organic Search** | Google, Bing, etc. without paid parameters |
| **Paid Search** | Google/Bing with gclid/msclkid |
| **Social** | Facebook, Twitter, LinkedIn, etc. |
| **Referral** | Other websites |
| **Email** | utm_medium=email |
| **AI Referral** | ChatGPT, Claude, Perplexity |

### AI Referral Tracking

Convultra uniquely tracks traffic from AI platforms:

| Platform | Referrer Pattern |
|----------|-----------------|
| ChatGPT | chatgpt.com |
| Claude | claude.ai |
| Perplexity | perplexity.ai |
| Google Bard/Gemini | bard.google.com |
| Bing Copilot | bing.com/chat |

View AI traffic in **Analytics → AI Referrals**

### Referrers

The Referrers table shows:
- Domain of referring site
- Number of visitors
- Conversion count & rate

### UTM Parameters

Break down traffic by UTM tags:

- **utm_source** - Traffic source (google, facebook)
- **utm_medium** - Marketing medium (cpc, email, social)
- **utm_campaign** - Campaign name
- **utm_term** - Paid search keywords
- **utm_content** - A/B test variations

## Top Pages

### Entry Pages

Pages where visitors land first:

| Column | Description |
|--------|-------------|
| Page | URL path |
| Visitors | Unique visitors entering |
| Bounce Rate | % leaving from this page |
| Conversion Rate | % converting after entering |

### Exit Pages

Pages where visitors leave:

| Column | Description |
|--------|-------------|
| Page | URL path |
| Exits | Number of session exits |
| Exit Rate | Exits / Pageviews |

### All Pages

All pages viewed, sorted by pageviews:

- Path
- Pageviews
- Unique pageviews
- Avg. time on page

## Privacy

Convultra analytics are privacy-friendly:

- ✅ No cookies for analytics
- ✅ Session-based, not cross-site
- ✅ No personal data stored
- ✅ IP addresses not logged
- ✅ GDPR/CCPA compliant

<Callout>
  Convultra does not require cookie consent banners for basic analytics.
</Callout>
