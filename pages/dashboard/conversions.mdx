# Conversions

Track, analyze, and manage your conversions in the Conversions dashboard.

## Conversion Overview

The Conversions page shows:

| Metric | Description |
|--------|-------------|
| **Total Conversions** | All conversion events |
| **Total Value** | Sum of conversion values |
| **Avg. Value** | Average conversion value |
| **Conversion Rate** | Conversions / Visitors |

| Column | Description |
|--------|-------------|

| Event | Event type (purchase, lead, etc.) |

## Conversion Types

### Standard Events

| Event | Typical Usage |
|-------|---------------|
| `purchase` | E-commerce transactions |
| `lead` | Form submissions |
| `signup` | Account registrations |
| `subscription` | Recurring signups |
| `download` | Content downloads |

### Custom Events

Any `track()` call can be a conversion:

```javascript
Convultra.track('demo_booked', {
  value: 100,
  demo_type: 'enterprise'
})
```

## Platform Forwarding

### Forwarding Status

Each conversion shows status per connected platform:

| Status | Meaning |
|--------|---------|
| ‚úÖ **Sent** | Successfully delivered |
| ‚è≥ **Pending** | In queue |
| üîÑ **Retrying** | Failed, retrying |
| ‚ùå **Failed** | Permanent failure |
| ‚ö™ **N/A** | Not applicable |

### Why "N/A"?

A conversion shows N/A for a platform when:
- No click ID for that platform (user didn't come from that ad)
- Platform not connected
- Event type not mapped

### Viewing Logs

**Conversions ‚Üí [Click conversion] ‚Üí View Forwarding Logs**

Detailed logs show:
- Timestamp
- Request sent
- Response received
- Error details (if any)

## Deduplication

Convultra prevents double-counting:

### Deduplication Methods

| Method | How It Works |
|--------|--------------|
| Order ID | Same orderId within 24h = duplicate |
| Event ID | Same eventId within 5min = duplicate |
| Click ID | Same click ID + value within 1h = duplicate |
| Fingerprint | Same fingerprint + value within 5min = duplicate |

### Viewing Duplicates

Duplicate events are logged but not counted:

**Conversions ‚Üí View Duplicates**
